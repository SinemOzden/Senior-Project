To run run the code with pycharm download all imported libraries (numpy, opencv, pyqt5,..).
To run the front user interface use "from InterfaceFront import Example" line from demo.py file, to run side use "from InterfaceDemoWithData import Example".
To select a video select edit configurations from toolbar and write the video name to the parameters part.
Then run the code. 
Notes: The demo code calss multiple files make sure every file .py file is in the same file. To learn more you can see the comments in the code.
